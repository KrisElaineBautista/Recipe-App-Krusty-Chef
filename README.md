# Krusty Chef Recipe App

This is a group project created as part of our course.

## Contributors
- Leader: Kris Elaine S. Bautista
- Member: Ron Jansen Dalida
- Member: Mary Rose Hagler

## Project Overview
Krusty Chef is a recipe application that allows users to:
- Browse and search recipes
- Add their own recipes
- Save favorites for easy access
- Login/Signup

## Ownership and Permission
All contributors share ownership of this project.  

## Technologies Used
- [Kotlin, Android Studio, Firebase]
